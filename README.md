# Bini Garage - Estética Automotiva

Site profissional para a Bini Garage, empresa especializada em estética automotiva.

## 🎨 Design

- **Cores**: Vermelho (#DC143C) e Preto (#000000)
- **Fontes**: Bebas Neue (títulos) e Inter (corpo do texto)
- **Estilo**: Moderno, elegante e profissional

## 📁 Estrutura do Projeto

```
binigarage/
├── index.html      # Página principal
├── style.css       # Estilos CSS
├── script.js       # Funcionalidades JavaScript
└── README.md       # Documentação
```

## 🚀 Funcionalidades

### Página Inicial
- Hero section com animação glow no título
- Navegação fixa com efeito scroll
- Design responsivo para dispositivos móveis

### Seções
- **Serviços**: Cards animados com 4 serviços principais
- **Sobre**: Informações sobre a empresa com estatísticas
- **Contato**: Formulário funcional com validação

### Funcionalidades JavaScript
- Menu hambúrguer para mobile
- Scroll suave entre seções
- Animações ao rolar a página
- Formulário de contato com validação
- Sistema de notificações
- Efeito parallax na hero section
- Formatação automática de telefone

## 🌐 Como Usar

1. **Abra o site**: Simplesmente abra o arquivo `index.html` em qualquer navegador moderno

2. **Navegação**: Use o menu superior ou role a página para explorar as seções

3. **Formulário de Contato**: 
   - Preencha todos os campos obrigatórios
   - Selecione o serviço desejado
   - Clique em "Enviar Solicitação"

4. **Design Responsivo**: O site se adapta automaticamente a diferentes tamanhos de tela
